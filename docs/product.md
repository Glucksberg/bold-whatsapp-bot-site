# Produto — Bold WhatsApp Bot V1

## Função
Bot de recepção e triagem da Bold no WhatsApp.

## Faz
- saúda o cliente
- entende motivo do contato
- coleta dados essenciais
- registra demanda
- confirma encaminhamento

## Não faz
- fechar venda
- aprovar orçamento
- substituir humano

## Tipos de demanda
- orçamento
- agendamento / disponibilidade
- dúvida geral
- falar com humano
- pós-atendimento
- parceria / comercial

## Campos da demanda
- id
- dataHora
- nome
- telefone
- empresa
- tipo
- resumo
- cidade
- dataDesejada
- urgencia
- responsavel
- status
- observacoes
